# Games Snake — All Hands Julio 🎮📼

Snake tipo NOKIA (LCD verde, tema submarino) para presentar los logros del
equipo de Monoku Games en el All Hands de Julio.

**Jugar:** https://m4r1l1.github.io/games-snake-allhands/

- Un solo archivo `index.html`, se abre en el navegador. Sin instalar nada.
- Controles: flechas o WASD. Pensado para proyectar en pantalla completa.
- 3 niveles (los logros del equipo) + pantalla de victoria con el equipo.

Ver `SPEC.md` para el detalle de cómo funciona.
